Shrink Wrap

Shrink wrap terrain under any object with a 3D physics collider.

Procedural Brushes

Terrain Former has customizable procedural falloff and perlin noise brushes.

Terrain Grids

Create massive and detailed terrains by working with a grid of terrains. Terrain Former has support for terrain grids across all of its features, including heightmap import/export, sculpting and painting.

Terrain Grid Wizard

Create a terrain grid without laboriously setting each parameter of each terrain with the Create Terrain Grid Wizard (found in "Game Object\3D Object\Create Terrain Grid"). Thers is also a "Statistics" dropdown that gives you an idea of how much memory the terrain grid will occupy as assets.

Heightmap Import/Export

Import heightmaps from any texture format Unity can import, and export to EXR with full support from terrain grids. Note that exporting to EXR is availble only in Unity 2017.1 and newer.

Flatten

Creates flat terrain based around three sub-modes; "Flatten", "Bridge", "Extend".

Paint Texture

Smooth out bumpy areas multiple times faster than Unity's builtin terrain tools.

Raise/Lower

The most basic of tools; raise and/or lower the terrain. Interactively raise or lower a specific part of the terrain with Ctrl+LMB.

Smooth

Smooth out bumpy areas multiple integers faster than Unity's builtin terrain tools.

Set Height

Sets the height of all terrain in the brush area based on the Set Height value. Shift+LMB will sample the height of the terrain the mouse in currently hovering over.

Unity Native

The entire interface and user experience is designed to feel like it's a native part of Unity, from the Unity-inspired tool icons to the preferences panel.
